1. Dependencies Used & Why
List all the packages and plugins you used in the project, along with a short explanation of why each dependency was necessary.
Example:
http: Used for making API calls.

2. Project Structure
Provide a brief explanation of your project’s folder structure and purpose of each main directory.
Example:
lib/

 ├── views/
 ├── controllers/
 ├── utils/
 ├── screens/

Explain what each folder contains.
3. App Screenshots
Add clear screenshots of your app showcasing the main features and UI.
4. App Video
Include a short video demonstration of your app (e.g., screen recording) showing the app’s flow and functionality.
